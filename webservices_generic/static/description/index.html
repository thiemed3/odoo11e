Odoo Generic Webservices Repository

This Module allows you to define several webservers entry point addresses for Odoo.

by Blanco Martín & Asociados

This is a technical module, intended to be the place where you can define your own or
third party services, with their authentication methods, so you can manage your
web services in one single place.

What to do next:


  • Enter your webservice server url
  • Define the auth method.
  • Define de auth key
  • Develop your own module, containing your procedures to take advantadge of the connection

    • Enhancements needed: (TO-DOs)


  • Add more auth methods.
  • Add more security options.
